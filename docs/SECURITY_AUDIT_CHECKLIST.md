# 🔒 BlockAI 安全审计清单

---

## 📋 合约安全检查

### 基础安全

- [ ] **重入攻击防护**
  - [ ] 使用 `ReentrancyGuard`
  - [ ] Checks-Effects-Interactions 模式
  - [ ] 状态更新在转账前完成

- [ ] **访问控制**
  - [ ] `onlyOwner` 修饰器正确使用
  - [ ] 敏感函数有权限检查
  - [ ] 所有权转移有安全机制

- [ ] **整数溢出/下溢出**
  - [ ] 使用 Solidity 0.8+（内置保护）
  - [ ] 或使用 SafeMath 库

- [ ] **前端运行防护**
  - [ ] 提交时设置滑点
  - [ ] 使用私有交易（可选）

### 业务逻辑安全

- [ ] **支付验证**
  - [ ] 检查 `msg.value` 是否足够
  - [ ] 正确处理找零
  - [ ] 防止重复支付

- [ ] **限额控制**
  - [ ] 总供应量上限检查
  - [ ] 每钱包限额检查
  - [ ] 时间窗口限制

- [ ] **白名单安全**
  - [ ] Merkle 证明验证正确
  - [ ] 防止重放攻击
  - [ ] 叶子节点唯一性

- [ ] **时间依赖**
  - [ ] 不使用 `block.timestamp` 做精确计时
  - [ ] 考虑矿工操纵可能
  - [ ] 设置合理时间窗口

### Gas 优化

- [ ] **存储优化**
  - [ ] 使用合适的数据类型（uint256 vs uint8）
  - [ ] 打包存储变量
  - [ ] 减少 SSTORE 操作

- [ ] **循环优化**
  - [ ] 避免无上限循环
  - [ ] 批量操作代替多次调用
  - [ ] 使用事件代替存储

- [ ] **内存优化**
  - [ ] 使用 `memory` 而非 `storage`
  - [ ] 缓存数组长度
  - [ ] 避免不必要的复制
